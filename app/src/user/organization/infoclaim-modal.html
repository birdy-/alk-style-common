Le Registre de commerce des sociétés dit RCS, est une base de données des personnes physiques ou morales dont l’activité ou la structure est commerciale. (Plus d'informations)


Le GLN (Global Location Number) identifie l'adresse d'une société ou d'un lieu physique/fonctionnel, typiquement un siège administratif, une filiale, un département comptable, un quai de déchargement, ... Les GLN sont surtout utilisés dans les messages EDI et GDSN pour identifier de manière unique les fournisseurs, centres de distribution, ... (Plus d'informations)
